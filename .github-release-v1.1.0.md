## ğŸš€ Quick Start

### Installation

Add to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/yourusername/RIOImageKit.git", from: "1.1.0")
]
```

### Basic Usage

```swift
import RIOImageKit

struct ContentView: View {
    let imageURL = URL(string: "https://example.com/image.jpg")!

    var body: some View {
        CachedAsyncImage(url: imageURL)
            .frame(width: 300, height: 300)
    }
}
```

### With Placeholder & Error Handling

```swift
CachedAsyncImage(url: imageURL) { state in
    switch state {
    case .idle, .loading:
        ProgressView()
    case .success(let image):
        Image(uiImage: image)
            .resizable()
            .aspectRatio(contentMode: .fit)
    case .failure:
        VStack {
            Image(systemName: "photo")
            Text("Failed to load")
        }
    }
}
```
